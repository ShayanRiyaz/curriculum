We want a dataframe with the following structure:



|          | Candidate1      | Candidate2      | ...  |
| -------- | --------------- | --------------- | ---- |
| positive | 321             | 23              | ...  |
| neutral  | 76              | 32              | ...  |
| negative | \# example data | \# example data | ...  |

Our dataframe `df` will do the trick: 

```
df = pd.DataFrame([], index=['positive', 'neutral', 'negative'])
```
